# StockLink Device Operations Manual

## Purpose

This manual explains the basic, everyday use of the StockLink device.

## 1. What the Device Is For

The device automates feeding and water control for the StockLink system. It also lets you run a feed immediately when needed and check the device status from the website.

## 2. Before Use

Make sure:

- The device has power.
- The device is connected to Wi-Fi.
- Feed and water are available.
- The device is installed correctly.

## 3. Starting the Device

When the device turns on:

1. Wait for it to finish starting up.
2. Make sure it connects to the network.
3. Confirm that it appears online in the website.

If the device does not come online, check the power supply, Wi-Fi connection, and physical installation.

## 4. Using the Device Normally

In normal use, the device will:

- Run feeding at the scheduled times.
- Keep the water system available.
- Follow the settings chosen on the website.

You do not need to control the device constantly. Most day-to-day changes are made from the website.

## 5. Checking Device Status

Use the website to check:

- Whether the device is online.
- Whether feed and water levels are okay.
- Whether any alerts are active.
- Whether a feeding schedule is enabled.

If something looks wrong, check the device first before making changes.

## 6. Running Feed Manually

Use a manual feed only when you need to give feed right away.

Steps:

1. Open the website.
2. Go to Schedule Management.
3. Enter the amount you want to feed.
4. Submit the manual feed request.
5. Wait for the device to complete the feed.

Only staff users can start a manual feed.

If the request does not work, check:

- Whether the device is online.
- Whether there is enough feed.
- Whether the amount entered is valid.

## 7. Scheduled Feeding

Scheduled feeding happens automatically when the schedule is enabled.

To make a schedule work properly:

- Give it a clear name.
- Pick the correct days.
- Set the correct time.
- Enter the right amount.
- Leave it enabled.

If feeding happens at the wrong time, check the timezone and schedule settings in the website.

## 8. Keypad Use

If your device has a keypad, it can be used for local input.

If the keypad does not respond correctly:

1. Check that it is connected properly.
2. Make sure no key is stuck.
3. Recheck the keypad settings if your system uses them.

## 9. Safety

Watch for these signs:

- The device stops responding.
- Feed does not dispense when expected.
- Water is not available when expected.
- The device reports a low battery or power issue.

If you see a power or battery warning, stop normal use and fix the issue before continuing.

## 10. Basic Troubleshooting

### Device is offline

- Check power.
- Check Wi-Fi.
- Restart the device if needed.

### Feeding does not happen

- Check that the schedule is enabled.
- Check the feed level.
- Check that the device is online.
- Make sure the current time is correct.

### Manual feed does not work

- Check that the amount is valid.
- Check that the device is online.
- Check that there is enough feed.

### Water does not dispense

- Check the water supply.
- Check that the system is powered.
- Check for a blockage or physical problem.

### Device seems unresponsive

- Wait a moment and try again.
- Check power and network.
- Restart the device if necessary.

## 11. Daily Checklist

Use this short checklist each day:

- Confirm the device is online.
- Check feed and water levels.
- Review active alerts.
- Confirm the feeding schedule is correct.
- Use manual feed only when needed.

## 12. When to Get Help

Get help if:

- The device will not power on.
- The device cannot connect to Wi-Fi.
- Feeding fails repeatedly.
- Water control fails repeatedly.
- You see repeated battery or power warnings.
